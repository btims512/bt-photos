# BT Photos — Benjamin Tims Photography

Personal photography website for Benjamin Tims (@benjammin.visuals), an Austin-based photographer with ties to Denver.

## About

A clean, dark, editorial-style portfolio site built with React and Vite. Features a full-page hero, filterable portfolio gallery, interactive flip-card services section, about section, and contact form.

## Services Featured

- **Portrait Sessions** — individuals, couples, families, and headshots
- **Events** — live music, solo artists, comedians, and full shows
- **Maternity & Newborn** — maternity sessions, newborn sessions, and bundles
- **Real Estate & Airbnb** — interior/exterior photography and walkthrough video

## Tech Stack

- [React](https://react.dev)
- [Vite](https://vitejs.dev)
- [Tailwind CSS](https://tailwindcss.com)
- [Lucide React](https://lucide.dev) — icons

## Getting Started

```bash
npm install
npm run dev
```
